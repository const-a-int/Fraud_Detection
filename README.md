# Fraud Detection API

API для обнаружения мошеннических транзакций на основе машинного обучения. Модель обучена на синтетических данных IBM и определяет вероятность мошенничества по характеристикам транзакции.

## 🚀 Возможности

- Определение вероятности мошенничества по транзакции
- Пакетная обработка нескольких транзакций
- Загрузка CSV файлов с транзакциями
- Оценка уровня риска (low/medium/high)
- REST API с документацией

## 🛠 Технологии

- **Python** 3.10+
- **FastAPI** - веб-фреймворк
- **Scikit-learn** - машинное обучение
- **XGBoost / Random Forest** - классификаторы
- **Pandas / NumPy** - обработка данных
- **Joblib** - сериализация моделей
